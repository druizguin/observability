# OpenObserve

OpenObserve (O2) is an open-source observability platform that provides a unified view of logs, metrics, and traces. It is designed to help developers and operators monitor and troubleshoot their applications effectively.

This demo requires API Key configured in .env file:

OPENOBSERVE_APIKEY

- Open O2 URL: [O2](http://localhost:5080/web/ingestion/recommended/kubernetes?org_identifier=default)
- Serach Opentelemetry  
- Take the token. Example: cm9vdEBleGFtcGxlLmNvbTpUU1hLTEJic0ViRXFI....
- Open .env file and set token in the variable OPENOBSERVE_APIKEY
